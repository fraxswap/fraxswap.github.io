What is Fraxswap?

Fraxswap is Frax Finance's permissionless AMM with an embedded TWAMM for long-term orders; the official docs describe a Uniswap V2-style constant-product core extended for time-weighted execution.

How Fraxswap works

  1. Open the appUse the official Frax swap route, connect a Web3 wallet, and select the network you want to use.
  2. Choose tokensPick the input and output assets, then review the displayed route, price impact, and slippage settings.
  3. Review timingUse a standard swap for immediate execution or review TWAMM order details when placing longer-term trades.
  4. Confirm on-chainApprove the token if required, submit the transaction, and verify the result in your wallet or explorer.

Before you swap

Built for time-weighted execution

The embedded TWAMM design lets long-term orders execute over time while interacting with the underlying AMM.

Familiar AMM base

The core follows a Uniswap V2-style constant-product model, with technical details available in the Frax technical specifications.

Frax ecosystem focus

The interface is suited for FRAX, FXS, and supported LP pairs across the deployments listed by Frax documentation.

Fraxswap FAQ

What is fraxswap?

fraxswap is Frax Finance's AMM interface for token swaps and TWAMM-style long-term orders, built around a Uniswap V2-style constant-product core.

Is fraxswap the official app?

This page is a routing page; use the official app link and verify the domain before connecting a wallet or signing any transaction.

Does fraxswap support TWAMM orders?

Yes. Frax documentation describes an embedded TWAMM for splitting long-term orders over time instead of executing the full amount at once.

Which networks are listed for fraxswap V2?

Frax documentation lists V2 deployments across networks including Ethereum, Fraxtal, Arbitrum, Avalanche, BSC, Fantom, Optimism, Polygon, Moonbeam, and Moonriver.

Do I need ETH to use fraxswap on Ethereum?

Yes. Ethereum transactions require ETH for gas, and other supported networks require their own native gas token.
