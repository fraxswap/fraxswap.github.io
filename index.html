What is Fraxswap?

Fraxswap is Frax Finance's permissionless AMM with an embedded TWAMM for long-term orders; the official docs describe a Uniswap V2-style constant-product core extended for time-weighted execution.

Fraxswap TWAMM orders

Fraxswap is searched specifically because it is not just another Uniswap-style swap page. The official Frax docs describe Fraxswap as a constant-product AMM with an embedded time-weighted average market maker, or TWAMM, for conducting large trades over longer periods. TWAMM-style execution can help split a large order over time instead of pushing the full amount through at once. Users should still check token pair liquidity, order duration, price impact, slippage, gas and whether they are placing a normal swap or a long-term order before signing.

Fraxswap fees and V2 pools

Fraxswap V2 introduced support for different LP fees per pool, according to the official contract-address documentation. That means there is no single Fraxswap fee that applies to every pair and chain. Fee impact also combines with pool liquidity, price movement, slippage and gas. Users comparing Fraxswap fees should inspect the live app quote for the specific FRAX, FXS or Frax ecosystem pair they want to trade, then compare the net received amount against another DEX route rather than relying on a generic fee percentage.

Fraxswap supported networks and contracts

Fraxswap V2 contract documentation lists deployments on Arbitrum, Avalanche, Binance Smart Chain, Ethereum, Fantom, Fraxtal, Moonbeam, Moonriver, Optimism and Polygon. The official docs also provide factory and router addresses for those networks. A user should not copy contract addresses from social posts or ads; use the Frax documentation or the official app. When switching chains, confirm the correct native gas token and verify whether the desired pair exists with enough liquidity on that chain.

Fraxswap FAQ

What is Fraxswap?

Fraxswap is Frax Finance's permissionless AMM with an embedded TWAMM for long-term orders. The core is based on a Uniswap V2-style constant-product model, but Fraxswap adds time-weighted execution features designed for large or sustained trades.

Does Fraxswap support TWAMM?

Yes. TWAMM support is the main reason Fraxswap stands out. The official docs describe Fraxswap as a constant-product AMM with an embedded time-weighted average market maker. Users still need to distinguish between a standard swap and any long-term order flow in the live interface.

Which chains is Fraxswap on?

The official Fraxswap V2 contract page lists deployments on Arbitrum, Avalanche, Binance Smart Chain, Ethereum, Fantom, Fraxtal, Moonbeam, Moonriver, Optimism and Polygon. Pair liquidity can differ significantly by chain, so the live quote matters more than the deployment list alone.

What tokens are usually swapped on Fraxswap?

Fraxswap is naturally associated with Frax ecosystem assets such as FRAX and FXS, plus paired assets on supported networks. Since Fraxswap is permissionless, available pairs can vary. Users should check the app for pool liquidity, token contracts and price impact before assuming a pair is active.

Is Fraxswap the same as Fraxlend?

No. Fraxswap is the AMM and TWAMM swap product. Fraxlend is a separate Frax Finance lending product. Searchers looking to swap FRAX, FXS or related assets should use the swap interface, while borrowing or lending research belongs to Fraxlend documentation.

Where are Fraxswap contract addresses?

Frax publishes Fraxswap V1 and V2 contract addresses in its official documentation. Use that page or the official app rather than addresses from social posts. For V2, the docs list factory and router addresses by chain, including Ethereum, Fraxtal, Arbitrum, Avalanche, BSC, Optimism and Polygon.
